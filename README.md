#Implementação de um agente capaz de jogar a primeira fase (1-1) de Super Mario Bros do NES

Trabalho realizado para a disciplina de Inteligência Artificial para Jogos do curso de Jogos Digitais da Unisinos. A ideia foi utilizar a biblioteca ML-Agents
para o motor Unity e treinar um modelo de IA capaz de completar a primeira fase do primero mundo de Super Mario Bros do console NES.

Como testar o agente no sistema operacional Windows:
  - Extrair o conteúdo do .zip 'WinBuild';
  - Rodar o executável 'Super Mario Bros.exe'.

  Ao fazer isso será possível ver o agente Mario jogar a fase múltiplas vezes em loop, ele nem sempre será capaz de completar a fase pois não foi treinado ao ponto
  de repetir com a mesma precisão sempre (o que deixa um pouco mais divertido de olhar! ><).
